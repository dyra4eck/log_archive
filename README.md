# log_archive

## Getting Started
1. **Clone the repository**
  ```
  git clone https://github.com/dyra4eck/log_archive.git
  cd log_archive
  ```

2. **Make the script executable**
  ```
  chmod +x log-archive
  ```

3. **Execute the script**
  ```
  ./log-archive /var/log
  ```
